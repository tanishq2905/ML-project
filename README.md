# Music Emotion Detection

## Overview

This project predicts the **mood of music** using machine learning based on emotional features.

## Features

* Data visualization of emotions
* Multiple ML models
* Model comparison
* Confusion matrices

## Models Used

* Logistic Regression
* Random Forest
* Neural Network
* PCA + Logistic Regression

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
